
## 🔷 Java Project : The "FinSafe" Transaction Validator

### Description
This project is an implementation of a basic banking application using core java concepts.  
This project enables the user to conduct basic banking transactions such as deposit, withdrawal, and viewing account balance.

---

### Features
- Deposit amount in account
- Withdraw amount from account
- Show balance
- Show past five transactions(Mini statement)
- Invalid input exceptions handling
- Transactions amount cannot exceed ₹20,000

---

### Concepts used
- Classes & Objects(Objects oriented programming)
- Arraylist
- Exceptions handling
- Loops & Switch case
